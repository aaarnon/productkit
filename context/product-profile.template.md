## Know Your Game

> The strategic game you're playing and how you keep score. When this is clear, prioritization becomes obvious.

| Field | Value |
|-------|-------|
| **The Game** | [What strategic game are we playing? e.g., "Engagement game - get users to spend more time" or "Conversion game - turn free users into paid"] |
| **How We Keep Score** | [1-3 primary metrics that define winning] |

**Examples of "games":**
- Engagement game — increase time spent, daily usage
- Conversion game — turn free users into paying customers
- Retention game — reduce churn at all costs
- Expansion game — grow into adjacent markets
- Acquisition game — maximize top-of-funnel growth
- Monetization game — increase revenue per user
